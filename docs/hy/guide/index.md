# Ի՞նչ է `climatenet-mcp`-ն

`climatenet-mcp`-ն Model Context Protocol սերվեր է՝ Հայաստանի ClimateNet բնապահպանական մոնիթորինգի տվյալները հարցնելու համար։

Այն տրամադրում է read-only MCP գործիքներ սարքեր գտնելու, մետատվյալներ դիտելու, վերջին չափումները կարդալու, գրաֆիկների համար պատրաստ ժամանակային շարքեր ստանալու և նույն ամսաթվերի միջակայքում սարքեր համեմատելու համար։

> `climatenet-mcp`-ն անկախ նախագիծ է և կապ չունի ClimateNet-ի հետ, հաստատված չէ նրա կողմից և չի սպասարկվում նրա կողմից։

## Մտածելակերպը

ClimateNet սարքերը հաղորդում են բնապահպանական տվյալներ, օրինակ՝ ջերմաստիճան, խոնավություն, ճնշում, մասնիկային նյութ, UV, լույս, քամի և անձրև։

`climatenet-mcp`-ն գործակալներին տալիս է կառուցվածքային գործիքների փոքր հավաքածու՝

- գտնել device ID-ները նախքան չափումներ հարցնելը
- դիտել մեկ սարքի մետատվյալները և սենսորների վիճակը
- ստանալ չափումները փաստաթղթավորված public API-ից
- վերադարձնել գրաֆիկի համար պատրաստ ժամանակային շարք մեկ metric-ի համար
- համեմատել մեկ metric մի քանի սարքերի միջև

Նպատակն է ClimateNet տվյալները հեշտ դարձնել MCP client-ներում ուսումնասիրելու համար՝ առանց գործակալներին ստիպելու էջեր քերել կամ API պարամետրեր կռահել։

## Խորհուրդ տրվող ընթացքը

Սկսեք սարքերը ցուցակելով՝

```text
list_devices({ "region": "Yerevan" })
```

Դիտեք սարքը՝

```text
get_device({ "device_id": 8 })
```

Ստացեք չափումներ կամ գրաֆիկային տվյալներ՝

```text
get_device_readings({
  "device_id": 8,
  "start_date": "2026-04-27",
  "end_date": "2026-04-28"
})

get_device_graph({
  "device_id": 8,
  "start_date": "2026-04-27",
  "end_date": "2026-04-28",
  "metric": "temperature"
})
```

Շարունակեք [Սկիզբ](/hy/guide/getting-started) էջում՝ MCP client միացնելու համար։
